# Exploratory Data Analysis (EDA)

---

## Project Overview

This project focuses on exploratory data analysis (EDA) using the analytical data warehouse developed in **Project 01 - Data Warehouse**.

The objective is to understand the structure, quality, and characteristics of the data before performing deeper analytical work. Through a series of SQL-based explorations, the project investigates dimensions, measures, date ranges, data distributions, and business performance indicators.

Exploratory analysis serves as a critical step in the analytics lifecycle by helping analysts identify patterns, anomalies, trends, and opportunities within the data.
