# SQL Analytics Portfolio Project

---

## Overview

This repository contains a collection of SQL Server projects that demonstrate practical applications of data warehousing, exploratory data analysis, and advanced analytics.

The projects follow an end-to-end analytics workflow, beginning with raw data ingestion and data warehouse development, followed by business-focused analysis and advanced analytical techniques to generate actionable insights.

The objective is to showcase SQL capabilities across the analytics lifecycle, including data modeling, ETL processes, business intelligence, and analytical problem-solving.

---

## Projects

### 01 - Data Warehouse

Design and build a modern SQL Server data warehouse using the Medallion Architecture (Bronze, Silver, and Gold layers).

**Key Topics**

* Data Architecture
* ETL Processes
* Data Quality Management
* Data Cleansing & Standardization
* Dimensional Modeling
* Star Schema Design

---

### 02 - Exploratory Data Analysis (EDA)

Analyze the curated warehouse data to uncover trends, patterns, and business insights.

**Key Topics**

* Business Metrics
* Customer Analysis
* Product Performance Analysis
* Revenue Analysis
* Trend Identification
* Descriptive Analytics

---

### 03 - Advanced Analytics

Apply advanced SQL techniques to solve analytical business problems and support data-driven decision-making.

**Key Topics**

* Customer Segmentation
* Cohort Analysis
* Ranking & Performance Analysis
* Window Functions
* Time-Based Analytics
* Advanced Business Insights
